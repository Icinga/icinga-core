As usual this is only a brief overview. Please take a look at the Changelog as well if you are missing something.

1.2.1. CHANGES

  • Icinga Classic UI adds on-demand filter popup and livesearch for better usability #852 #1657

  • custom vars are now visible in Icinga Classic UI. Filter them using 'exclude_customvar_{name,value}' #4390

  • remove slahistory, timedevents functionality #4363 #4482

  • IDOUtils clean_{config,realtime}_tables_on_core_startup config options DEPRECATED #4791

  • External Business Process Cronk removed from Icinga Web tarball #4818

  • All Flash components removed from the Icinga Web tarball. (#3474 #4806)

  • Icinga Web 1.10 requires an IDOUtils schema version of >= 1.10

  • Icinga Web and IDOUtils schema upgrades to 1.10 required.

1.2.2. Core

  • allow exclude (!) to work for regexp matches as well #3720

  • add check source to checkresult/status data #4709

  • multiple idomod modules: only first gets data from registered callback functions #4199

  • add manpages from debian upstream #4055

  • add vim highlight script for Icinga syntax #4150

1.2.3. Classic UI

  • add display filter modification for status.cgi #852

  • added live search #1657

  • show custom variables in extended info #4390

  • allow to exclude custom variable name/value based on pattern #4390

  • added option for partial servicegroups #2740

  • show check source in extinfo #4709

1.2.4. IDOutils

  • add a table trimming option for 'downtime history' #3989

  • allow to disable customvar status updates with dump_customvar_status=0|1 in idomod.cfg #4479

  • add object_id (host, service) to logentries status updates/state changes #4754

  • remove slahistory, timedevents functionality #4363 #4482

1.2.5. Icinga-Web

  • menu categories split and layout changed #4807 #4691

  • all problem cronks for hosts, services and split view #4167

  • instance cronk is more useful and disable all notifications with expire time #3956 #3888

  • delete all comments of a host/service #2636

  • add state_type shown in problems to all grids #3529

1.2.6. Icinga-Reporting

  • minor sql fixes for templates in postgresql #1880

1.2.7. Docs

  • add samples files for reference #4422

  • update pnp screenshots, explain icinga web expander icon #4527 #4288

  • add hint to views.xml config file #4690

  • add "show_partial_servicegroups" to cgi.cfg #4730

  • add "dump_customvar_status" to idomod.cfg #4705

  • remove enable_sla, timedevent* from idoutils docs #4487

  • update icinga web troubleshooting for timezones #4554
